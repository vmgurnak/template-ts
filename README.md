# TypeScript Static Project (No Bundler)

Small frontend project using **TypeScript without a bundler**.
TypeScript is compiled to plain JavaScript using `tsc`, and the result is deployed as static files.

## 🚀 Tech Stack

- TypeScript
- HTML5
- CSS3
- ES Modules
- GitHub Pages / Vercel (Static Hosting)

## 📁 Project Structure

src/ – TypeScript source files
dist/ – compiled JavaScript (used by browser)

## 🛠 Development

npm init (создаёт проект)
npm install typescript --save-dev (устанавливает TypeScript как dev-зависимость)
npx tsc --init (Создаёт файл tsconfig.json , cоздаёт файл tsconfig.json)

npm install (создаёт проект)
npm run watch

## 📦 Build

npm run build
